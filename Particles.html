Моддинг Частиц

Частицы - это специальные объекты, используемые другими объектами для создания визуальных эффектов, таких как молния, снег и т.д.
Частицы определяются в файлах .asset, которые находятся в /Hearts of Iron IV/gfx/particles/.

Добавление света

Огни - это особые типы частиц, на которые ссылаются в сущностях, например, вспышка дула пистолета во время анимации боя. Вот общий пример: 

light = {
    name = "name_of_light"
    
    color = {
        r = { <float> }
        g = { <float> }
        b = { <float> }
    }
    
    intensity = <float>, fade
    radius = <float>
    falloff = <float>
    
    position = {
        x = <float>
        y = <float>
        z = <float>
    }
    
    animation = {
        name = "name_of_animation"
        start = <float>
        duration = <float>
        repeat = yes
        op = <type>
        minValue = <float>
        maxValue = <float>
        curve = {
            <?>
        }
    }
}

name - название светильника.
color - цвет света. Значения принимаются в десятичном формате RGB.
intensity - интенсивность света, а второе значение - затухает ли он.
radius - радиус света.
falloff - градиент падения света.
position - положение света относительно объекта, использующего его.
animation - анимация, которая будет воспроизводиться для света.

Добавление света

name - имя анимации.
start - начальная позиция кривой.
duration - продолжительность анимации.
repeat определяет, будет ли анимация повторяться.
op определяет способ смешивания нескольких источников. Может быть мультипликативным (MUL), аддитивным (ADD) или абсолютным (ABS).
minValue - минимальное значение на кривой.
maxValue - максимальное значение на кривой.
curve - кривая, используемая для света.
