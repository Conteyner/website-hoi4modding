Постэффекты - это эффекты цветокоррекции, которые применяются к освещению в определенных областях (объемах) на карте. Они используются для тонирования освещения в областях, где освещение заметно отличается, например, в пустынях.

Все постэффекты находятся в файле /Hearts of Iron IV/gfx/posteffect_volumes.txt.

Значения постэффекта

Запись значений постэффекта соответствует этому формату: 


posteffect_values = {
    name = <name>
    inherit = <name>
    
    lut = <path>
    
    hdr_min_adjustment = <float>
    hdr_max_adjustment = <float>
    
    BLOOM_WIDTH = <float>
    BLOOM_SCALE = <float>
    BRIGHT_THRESHOLD = <float>
    
    tonemap_middlegrey = <float>
    
}

name - имя записи значений постэффекта.
inherit заставляет значения для текущего постэффекта наследоваться от указанного постэффекта.
lut - путь к карте тонов, используемой для этой записи.
hdr_min_adjustment определяет минимальную корректировку, используемую в эффекте HDR.
hdr_max_adjustment определяет максимальную корректировку, используемую в эффекте HDR.
BLOOM_WIDTH определяет ширину эффекта bloom.
BLOOM_SCALE определяет масштаб эффекта расцветки.
BRIGHT_THRESHOLD определяет порог эффекта расцветки.
tonemap_middlegrey определяет средний серый цвет, используемый в тоновой карте.

Объемы

Позиция oбъемa

Запись объема определяет область на карте, в которой применяются значения постэффекта. Она имеет следующий формат: 

posteffect_volumes = {
    posteffect_volume = {
        name = <name>
        posteffect_values_day = <name>
        posteffect_values_night = <name>
        posteffect_values_day_winter = <name>
        posteffect_values_night_winter = <name>
        
        position = {
            <x>
            <y>
            <z>
        }
        size = {
            <width>
            <height>
            <depth>
        }
        fade_distance = <float>
    }
}

name - имя записи объема.
posteffect_values_day - запись значений постэффекта для использования в течение дня.
posteffect_values_night - запись значений постэффекта для использования в ночное время.
posteffect_values_day_winter - запись значений постэффекта для использования днем в зимние месяцы.
posteffect_values_night_winter - запись значений постэффекта для использования ночью в зимние месяцы.
position определяет местоположение центра объема.
size определяет размер поля объема.
fade_distance определяет расстояние затухания для значений постэффекта.

Высота oбъемa

Запись объема высоты задает высоту, на которой применяются значения постэффекта. Он имеет следующий формат: 

posteffect_volumes = {
    posteffect_height_volume = {
        name = <name>
        posteffect_values_day = <name>
        posteffect_values_night = <name>
        
        height = <float>
        fade_distance = <float>
    }
}

name - имя записи объема.
posteffect_values_day - запись значений постэффекта для использования в течение дня.
posteffect_values_night - запись значений постэффекта для использования в ночное время.
height определяет, на какой высоте будут применяться значения.
fade_distance определяет расстояние затухания для значений постэффекта.

Советы

Используйте консольные команды: PostEffectVolumes.Enabled и PostEffectVolumes.Draw при работе с постэффектами. В настоящее время только для разработчиков.
Вы можете перезагрузить постэффекты с помощью команды reload posteffectvolumes в консоли.
