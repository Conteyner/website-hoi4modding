Графические активы - это файлы изображений, которые были определены в файле .gfx. Это связывает изображение с именем сценария, на который затем ссылаются файлы интерфейса .gui.

Файлы графических активов можно найти в /Hearts of Iron IV/interface/*.gfx.

Обратите внимание, что для многих активов, таких как определения активов технологий и оборудования,
изображение может быть указано для определенного тега, т.е. GFX_ENG_<имя>, который отображает актив только для указанного тега.
Это переопределит базовый актив с тем же именем.
Это также относится к графической культуре,
т.е. GFX_african_<имя>.

Sprite Types

spriteType

Запись актива spriteType и textSpriteType следует этому формату: 

spriteTypes = {
    spriteType = {
        name = "GFX_<name>"
        texturefile = <path>
        noOfFrames = <int>
        effectFile = <path>
        
        allwaystransparent = <bool>
        legacy_lazy_load = <bool>
        transparencecheck = <bool>
        
        animation = {
            animationmaskfile = <path>
            animationtexturefile = <path>
            animationrotation = <float>
            animationlooping = <bool>
            animationtime = <float>
            animationdelay = <float>
            animationblendmode = <mode>
            animationtype = <type>
            animationrotationoffset = { 
                x = <float>
                y = <float>
            }
            animationtexturescale = {
                x = <float>
                y = <float>
            }
            animationframes = {
                <int> <int># etc
            }
        }
    }
}

имя - это имя, которое вы дали активу. Оно может иметь префикс с различными другими сегментами, например, idea для идейных образов и т.д.

texturefile - это путь к изображению, основанный на /Hearts of Iron IV/, то есть вы включаете только папки внутри этой папки, а не полный путь к файлу.

noOfFrames используется для изображений с несколькими кадрами внутри, например, GFX_unit_level имеет пять изображений внутри для каждого из изображений уровня подразделения, и поэтому в определении актива будет установлено noOfFrames = 5.

effectFile используется для определения состояния эффекта для применения к изображению, т.е. эффект нажатия кнопки для изображения кнопки. Эффекты находятся в файле /Hearts of Iron IV/gfx/FX/*.lua. Для добавления нескольких эффектов можно задать несколько.

allwaystransparent определяет, можно ли нажимать на изображение и взаимодействовать с ним. Если да, то изображение не будет иметь всплывающей подсказки и на него нельзя будет нажать при использовании в игре.

legacy_lazy_load определяет, будет ли изображение загружаться.

transparencecheck определяет, будет ли изображение ограничено своим альфа-каналом относительно щелчков игрока. В противном случае границами являются размеры изображения.

область анимации используется для добавления анимации к изображению. Он использует следующие атрибуты:

animationmaskfile управляет изображением маски для анимации.

animationtexturefile управляет цветным изображением, используемым для анимации.

animationrotation управляет вращением анимационного изображения. По умолчанию он равен -90 по часовой стрелке.

animationlooping контролирует, зацикливается ли анимация.

animationtime управляет продолжительностью анимации. Она измеряется в секундах.

animationdelay управляет задержкой между циклами анимации. Она измеряется в секундах.

animationblendmode управляет режимом наложения для анимационного изображения. Это может быть режим добавления, умножения или наложения.

animationtype управляет типом анимации. Это может быть прокрутка, вращение или пульсация.

animationrotationoffset управляет смещениями x и y для поворота.

animationtexturescale управляет масштабом анимационного изображения.

animationframes устанавливает количество кадров в анимационном изображении.

Имейте в виду, что максимальный размер файла спрайта .dds составляет около 16 МБ.

frameAnimatedSpriteType

Запись актива frameAnimatedSpriteType имеет следующий формат: 

spriteTypes = {
    frameAnimatedSpriteType = {
        name = "GFX_<name>"
        texturefile = "<path>"
        noOfFrames = <int>
        effectFile = "<path"
        
        animation_rate_fps = <int>
        looping = <bool>
        play_on_show = <bool>
        pause_on_loop = <float>
        
        allwaystransparent = <bool>
    }
}

имя - это имя, которое вы дали активу. Оно может иметь префикс с различными другими сегментами, например, idea для идейных образов и т.д.

noOfFrames используется для изображений с несколькими кадрами внутри, например, GFX_unit_level имеет пять изображений внутри для каждого из изображений уровня подразделения, и поэтому в определении актива будет установлено noOfFrames = 5.

effectFile используется для определения состояния эффекта для применения к изображению, т.е. эффект нажатия кнопки для изображения кнопки. Эффекты находятся в файле /Hearts of Iron IV/gfx/FX/*.lua. Для добавления нескольких эффектов можно задать несколько.

animation_rate_fps - скорость воспроизведения анимации.

looping определяет, зацикливается ли анимация.

play_on_show определяет, начинает ли анимация воспроизводиться, когда она видна.

pause_on_loop определяет длительность паузы после цикла анимации.

allwaystransparent определяет, будет ли изображение ограничено своим альфа-каналом в отношении щелчков игрока. В противном случае границами являются размеры изображения.

progressbartype

Запись актива типа progressbartype используется для прогресс-баров. Он имеет следующий формат: 

spriteTypes = {
    progressbartype = {
        name = "GFX_<name>"
        textureFile1 = "<path>"
        textureFile2 = "<path>"
        color = { <r> <g> <b> }
        colortwo = { <r> <g> <b> }
        size = { 
            x = <int>
            y = <int>
        }
        effectFile = <path>
        
        horizontal = <bool>
    }
}

имя - это имя, которое вы дали активу. Оно может иметь префикс с различными другими сегментами, например, idea для идейных образов и т.д.

textureFile1 - это изображение, которое будет использоваться для индикатора выполнения.

textureFile2 - это изображение, которое будет использоваться для фона индикатора выполнения.

color определяет непрогрессирующий цвет полосы. Это десятичный RGB.

colortwo определяет прогрессирующий цвет полосы. Это десятичный RGB.

size определяет размер секции прогресс-бара по высоте и ширине.

effectFile используется для определения состояния эффекта для применения к изображению, т.е. эффект нажатия кнопки для изображения кнопки. Эффекты находятся в файле /Hearts of Iron IV/gfx/FX/*.lua. Для добавления нескольких эффектов можно задать несколько.

horizontal определяет, будет ли прогресс-бар горизонтальным или нет. По умолчанию полоса горизонтальна.

corneredTileSpriteType

Запись актива corneredTileSpriteType используется для текстур, которые укладываются на плитку, чтобы заполнить размер элемента интерфейса, в котором они используются. Она имеет следующий формат: 

spriteTypes = {
    corneredTileSpriteType = {
        name = "GFX_<name>"
        texturefile = "<path>"
        noOfFrames = <int>
        size = {
            x = <int>
            y = <int>
        }
        borderSize = {
            x = <int>
            y = <int>
        }
        effectFile = <path>
        
        allwaystransparent = <bool>
        tilingCenter = <bool>
        
        looping = <bool>
        animation_rate_spf = <int>
    }
}

имя - это имя, которое вы дали активу. Оно может иметь префикс с различными другими сегментами, например, idea для идейных образов и т.д.

texturefile - это путь к изображению, основанный на /Hearts of Iron IV/, то есть вы включаете только папки внутри этой папки, а не полный путь к файлу.

noOfFrames используется для изображений с несколькими кадрами внутри, например, GFX_unit_level имеет пять изображений внутри для каждого из изображений уровня подразделения, и поэтому в определении актива будет установлено noOfFrames = 5.

size определяет размер клетки.

borderSize определяет размер границы.

effectFile используется для определения состояния эффекта для применения к изображению, т.е. эффект нажатия кнопки для изображения кнопки. Эффекты находятся в файле /Hearts of Iron IV/gfx/FX/*.lua. Для добавления нескольких эффектов можно задать несколько.

allwaystransparent определяет, будет ли изображение ограничено своим альфа-каналом в отношении щелчков игрока. В противном случае границами являются размеры изображения.

tilingCenter определяет, используется ли для черепицы центр изображения, а не левый верхний угол.

looping определяет, зациклена ли анимация.

animation_rate_spf определяет частоту кадров, с которой воспроизводится анимация.

animatedmaptext

Запись актива animatedmaptext используется для текста, который появляется на карте. Он имеет следующий формат: 

objectTypes = {
    animatedmaptext = {
        name = <name>
        speed = <float>
        textblock = {
            text = <string>
            color = { <r> <g> <b> }
            font = <path>
            position = { 
                x = <int>
                y = <int>
            }
            size = {
                x = <int>
                y = <int>
            }
            format = <type>
            cull_distance = <float>
        }
    }
}

name - это название текста карты. Невозможно добавить пользовательские.

скорость - время затухания текста.

textblock управляет атрибутами отображаемого текста.

textcolors

Запись textcolors определяет цвета текста, используемые в локализации с помощью символа §. Она имеет следующий формат: 

bitmapfonts = {
    textcolors = {
        <symbol> = { <r> <g> <b> }
    }
}

символ - это символ, используемый в локализации, т.е. H.

RGB - это цвет, который применяется в символе. Это целочисленный RGB.

bitmapfont

Запись bitmapfont определяет шрифт, используемый в игре. Она имеет следующий формат: 

bitmapfonts = {
    bitmapfont = {
        name = <name>
        path = <path>
        color = <hex>
        border_color = <hex>
        fontfiles = {
            <path>
            <path>
        }
        textcolors = {
            <symbol> = { <r> <g> <b> }
        }
        cursor_offset = {
            <int>
            <int>
        }
    }
}

name - ссылочное имя шрифта для других графических файлов.

path - путь к шрифту. Шрифты находятся в /Hearts of Iron IV/gfx/fonts/.

color - это цвет шрифта по умолчанию. Он представляет собой шестнадцатеричный код RGB.

border_color - цвет границы шрифта. Это шестнадцатеричный код RGB.

fontfiles используется, когда для одного шрифта есть несколько изображений шрифтов, например, для поддержки английского и кириллицы. Он используется вместо path.

textcolors изменяет глобальные текстовые цвета для шрифта, в который он вложен.

cursor_offset смещает текстовый курсор для шрифта. Определяет позицию x, а затем позицию y.

bitmapfont_override

Запись bitmapfont_override используется для переопределения шрифта для определенной локали. Она имеет следующий формат: 

bitmapfonts = {
    bitmapfont_override = {
        name = <name>
        fontfiles = {
            <path>
            <path>
        }
        languages = {
            <language>
            <language>
        }
    }
}

name - имя шрифта, который нужно переопределить.

fontfiles используется, когда для одного шрифта есть несколько изображений шрифтов, например, для поддержки English и Cryllic. Он используется вместо path.

languages определяет, для каких языков применяется это переопределение.
