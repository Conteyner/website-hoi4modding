Определения звуков находятся в /Hearts of Iron IV/sound. Звуковой файл должен быть сохранен как WAV-файл в стерео, как моно канал, с частотой 44100 Гц и как 32-битный плавающий файл. Все файлы звуковых определений должны быть сохранены как файлы .asset.

Звук

Звуковая запись используется для определения звука. Она имеет следующий формат: 

sound = {
    name = <name>
    file = <path>
    always_load = <bool>
    volume = <float>
}

name - имя звукового определения, на которое ссылаются другие файлы.

file - путь к звуковому файлу, относительно папки звуков Hearts of Iron IV.

always_load определяет, будет ли звук загружаться всегда.

volume определяет громкость звука.

Моддинг звука

Запись звукового эффекта используется для определения звукового эффекта. Она имеет следующий формат: 

soundeffect = {
    name = <name>
    falloff = <name>
    sounds = {
        sound = <name>
    }
    
    loop = <bool>
    is3d = <bool>
    random_sound_when_looping = <bool>
    
    max_audible = <int>
    max_audible_behaviour = <type>
    
    volume = <float>
    fade_in = <float>
    fade_out = <float>
    
    delay_random_offset = {
        <float>
        <float>
    }
    playbackrate_random_offset = {
        <float>
        <float>
    }
    volume_random_offset = {
        <float>
        <float>
    }
}

name - имя звукового эффекта.
falloff - запись falloff, используемая звуковым эффектом.
sounds - звуковые записи, используемые звуковым эффектом.
loop определяет, зацикливается ли звуковой эффект.
is3d определяет, должен ли звуковой эффект использовать 3D-звук.
random_sound_when_looping определяет, выбирает ли звуковой эффект случайный звук при зацикливании, а не выбирает его итеративно.
max_audible определяет максимальное количество экземпляров звукового эффекта в один момент времени.
max_audible_behaviour определяет, что произойдет, если количество экземпляров превысит max_audible. Это всегда отказ.
volume определяет громкость звукового эффекта.
fade_in определяет продолжительность затухания звукового эффекта.
fade_out определяет продолжительность затухания звукового эффекта.
delay_random_offset определяет минимальное и максимальное случайное смещение задержки между звуковыми циклами.
playbackrate_random_offset определяет минимальное и максимальное случайное смещение скорости воспроизведения между звуковыми циклами.
volume_random_offset определяет минимальное и максимальное случайное смещение громкости между звуковыми циклами.

Falloff

Записи Falloff определяют атрибуты падения для звука. Они добавляются в звуковые эффекты и имеют следующий формат: 

falloff = {
    name = <name>
    
    min_distance = <float>
    max_distance = <float>
    height_scale = <float>
}

name - имя записи falloff.
min_distance - минимальное расстояние до применения falloff, т.е. максимальная громкость.
max_distance - максимальное расстояние до того, как звук не будет слышен.
height_scale - скаляр для высоты между источником звука и камерой игрока.

Категории

Звуковые эффекты могут быть помещены в категории звуков, к которым применяется определенный компрессор. Категории имеют следующий формат: 

category = {
    name = <name>
    soundeffects = {
        <name>
    }
    compressor = {
        enabled = yes
        pregain = <float>
        postgain = <float>
        ratio = <float>
        threshold = <float>
        attacktime = <float>
        releasetime = <float>
    }
}

name - название категории звуковых эффектов.
soundeffects - список звуковых эффектов, принадлежащих категории.
compressor - компрессор, используемый для данной категории, подробнее об атрибутах см. ниже.

Компрессоры

Существует два глобальных компрессора: master_compressor используется для звуков, а music_compressor - для музыки. Отдельные компрессоры могут быть определены для звуков в категориях.
Компрессоры используют следующий формат: 

<name> = {
    pregain = <float>
    postgain = <float>
    ratio = <float>
    threshold = <float>
    attacktime = <float>
    releasetime = <float>
}
