Моддинг портретов

Портреты, используемые случайными лидерами стран и военачальниками, указаны в /Hearts of Iron IV/portraits. Необходимо указать файлы в /Hearts of Iron IV/interface/_random_portraits.gfx, иначе игра не будет отображать изображения.

Портреты

Портреты используются для назначения пула изображений определенным группам стран или конкретной стране, которые затем будут использованы, когда игрок создаст военного лидера или через эффект будет заменен лидер страны или военный лидер.
Это происходит в следующем формате: 

default = { 
    # ... 
}
continent = {
    name = <continent> 
     # ... 
}
 = { 
    # ... 
}

default - это запасные портреты, используемые, когда ни одно другое определение не соответствует текущей области видимости.
continent определяет портреты для каждого континента, то есть они применяются к любой нации со столицей на указанном континенте.
tag определяет портреты для указанного тега, переопределяя определение континента, если оно также соответствует одному из них.
Фактические определения портретов выглядят следующим образом: 

male = {
    
}
female = {
    
}
    
army = {
    male = {
        
    }
    female = {
        
    }
}
navy = {
    male = {
        
    }
    female = {
        
    }
}
political = {
     = {
        male = {
            
        }
        female = {
            
        }
    }
}

Можно использовать несколько путей, чтобы добавить пул портретов к каждому определению.
Верхний мужской и женский пути могут быть использованы для любого типа лидера.
Пути армии используются для фельдмаршалов и командиров корпусов.
Пути флота используются для командующих флотом.
Политические пути используются для лидеров стран с указанными идеологиями.
